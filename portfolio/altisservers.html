Prophecy Gaming





Github Link

A website fully designed then developed by me as a gaming server ranking and viewing list

Users could submit their server and have them display on the website where others could vote on them

Users would log in via their Steam account using the Steam API

Back-End





Written in PHP5 and using the CodeIgnitor framework

Several different API's were utilised for this project, Steam, GameQuery and GeoIP

User accounts were managed based on their steamID, users could manage servers which they had posted
